## Findings vs. Last Week (April 9 → April 13)

### 🔴 P1 — DataDome Blocking Googlebot (PERSISTING — Week 2)
- **Issue:** All requests using `Googlebot/2.1` UA return 403 + DataDome HTML challenge. robots.txt returns HTML, not text/plain. ALL pages 403 for crawler UAs.
- **Impact:** HIGH — Googlebot may be failing to crawl new/updated pages. If Googlebot encounters consistent 403s, it backs off crawl frequency. Combined with a platform migration, this is a meaningful indexation risk.
- **Evidence:** `curl robots.txt -A "Googlebot/2.1"` → HTTP 403, DataDome challenge HTML. Response header: `x-datadome: protected`
- **Fix:** DealerOn must whitelist verified crawler IPs in DataDome. This should be standard for any automotive dealer platform — escalate directly to DealerOn support if not resolved.
- **Priority:** P1 — immediate escalation to DealerOn

### 🔴 P1 — Platform Migration Uncertain (NEW THIS WEEK)
- **Issue:** Expected migration to DealerOn by April 10. As of April 13, server header shows `Carbase 1.0`. DealerOn migration either incomplete, delayed, or DNS hasn't propagated globally.
- **Impact:** HIGH — if still on Carbase, SEO improvements planned for DealerOn template haven't been applied. If partial migration, risk of split canonicals, redirect chain breakage, or inconsistent crawl behavior.
- **Evidence:** Every HTTP response includes `server: Carbase 1.0`
- **Fix:** Confirm migration status with DealerOn. If complete, they may just not be stripping the `Carbase` server header (cosmetic only). If not complete, get an updated launch date and push DataDome fix to the top of their pre-launch checklist.
- **Priority:** P1

### 🟡 P2 — Mixed-Case Blog URLs Still in Sitemap (PERSISTING)
- **Issue:** Blog articles still indexed with mixed case: `/2019-Chevrolet-Equinox-VS-2019-Ford-Escape-IA`, `/SUV-For-Every-Stage-Of-Life-IA`, `/Why-Buy-a-Toyota-Crossover-Carroll-IA`
- **Impact:** MEDIUM — Google treats URL paths as case-sensitive; these pages may get duplicate indexed or split link equity with lowercase equivalents
- **Evidence:** Sitemap as of 2026-04-13 still includes these URLs unchanged
- **Fix:** If DealerOn migration is complete, verify these blog URLs exist and redirect mixed-case versions to lowercase equivalents (301). If on Carbase still, add this to DealerOn migration checklist.
- **Priority:** P2

### 🟡 P2 — Outdated Model Year Content Still Live (PERSISTING)
- **Issue:** Sitemap includes 2020/2021 model year pages: `/2020-toyota-tundra-capabilities-carroll-ia`, `/2020-toyota-rav4-safety-caroll-ia` [note: "caroll" typo in URL], `/2021-toyota-rav4-spotlight-carroll-ia`, `/2020-Chevrolet-Silverado-1500-Estherville-IA`
- **Impact:** MEDIUM — stale content wastes crawl budget, signals low E-E-A-T freshness, and may confuse buyers
- **Evidence:** All present in April 13 sitemap
- **Fix:** Noindex or redirect these to current model year equivalents. The `/2020-toyota-rav4-safety-caroll-ia` also has a typo in the slug ("caroll" vs "carroll") — fix that too.
- **Priority:** P2
